# vCard-QR
Create a vCard QR code

## Dependencies:
Script requires `qrencode` be installed.

## Use the script
The script takes 5 arguments:
1. First Name
2. Surname
3. Phone Number
4. Email Address
5. Output file name (SVG)

## Execute the script:
`./vCard-qr.sh`

## Check the file works
`open <filename>.svg`

Scan the QR code with a smartphone to check the vCard loads.
